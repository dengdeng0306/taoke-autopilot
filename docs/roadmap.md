# 项目文档

## 模块说明

- **collector** - 选品采集：通过浏览器自动化在选品广场抓取商品数据
- **linker** - 推广链接：批量生成淘口令/推广链接（待实现）
- **tracker** - 收益追踪：从结算中心提取收益数据（待实现）
- **reporter** - 报表生成：将采集数据生成CSV/HTML日报
- **monitor** - 风险监控：佣金变动检测+链接有效性检查+告警推送
